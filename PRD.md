# GomokuBattle 五子棋对战平台 PRD文档

## 一、产品背景与解决的痛点

### 核心痛点（基于全网调研）
| 排名 | 痛点 | 解决方案 |
|------|------|----------|
| 1 | **广告骚扰** - 对弈中弹出广告导致游戏中断判负 | 零广告承诺，纯游戏体验 |
| 2 | **操控差** - 落子位置偏移，误触频发 | 精确落子反馈，清晰网格点击 |
| 3 | **匹配不公平** - 新手匹配到高手 | 段位匹配系统 |
| 4 | **功能单一** - 无好友、无法观战 | 房间对战，观战系统 |
| 5 | **Bug频发** - 禁手判定错误、闪屏卡顿 | 稳定的后端逻辑，前端Canvas优化 |
| 6 | **无法悔棋** | 提供人机悔棋选项 |
| 7 | **获胜连线不显示** | 醒目直线标注获胜五子 |

## 二、用户画像

- **目标用户**：办公室白领、学生、休闲玩家
- **使用场景**：工作间隙放松、朋友对战、在线匹配
- **核心需求**：快速开始、流畅对战、无广告干扰

## 四、页面路由清单

1. **/** - 首页（仪表盘，显示用户信息、快速开始入口）
2. **/game/ai** - 人机对战
3. **/game/room** - 房间列表/创建房间
4. **/game/play/{room_id}** - 对局页面（支持AI/房间对战）
5. **/rankings** - 排行榜

## 五、详细功能需求

### P0 核心功能
1. **Canvas棋盘绘制** - 15x15标准棋盘，清晰网格线
2. **落子交互** - 点击精确落子，视觉反馈
3. **胜负判定** - 横/竖/斜五子连珠判定
4. **获胜连线高亮** - 醒目红色直线标注
5. **AI对战** - 基础AI算法（评分策略）
6. **房间匹配** - 创建房间/加入房间
7. **实时对战** - WebSocket通信

### P1 增强功能
1. **悔棋功能** - 人机模式下允许悔棋
2. **倒计时** - 每步30秒超时判负
3. **段位系统** - 新手/业余/高手/大师
4. **排行榜** - 按胜率/胜场排名

### P2 优化功能
1. **历史对局** - 查看最近10局记录
2. **音效反馈** - 落子音效
3. **最后落子标记** - 显示对方最后落子位置

## 六、技术架构

### 后端
- **框架**：FastAPI + Uvicorn
- **实时通信**：WebSocket（fastapi websockets）
- **数据库**：SQLite
- **ORM**：SQLAlchemy 2.0

### 前端（PC自适应）
- **UI框架**：Bootstrap 5
- **交互**：jQuery 3.7
- **绘图**：Canvas API
- **实时**：原生WebSocket

### API设计

#### 认证接口
- `POST /api/auth/register` - 注册
- `POST /api/auth/login` - 登录

#### 游戏接口
- `POST /api/game/ai/start` - 开始人机对战
- `POST /api/game/ai/move` - 提交AI对战落子
- `POST /api/game/ai/undo` - 悔棋请求

#### 房间接口
- `GET /api/room/list` - 获取房间列表
- `POST /api/room/create` - 创建房间
- `POST /api/room/join/{room_id}` - 加入房间
- `WebSocket /ws/game/{room_id}` - 实时游戏通信

#### 排行榜
- `GET /api/rankings` - 获取排行榜

## 七、数据库设计

### User表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer PK | 用户ID |
| username | String | 用户名 |
| password_hash | String | 密码哈希 |
| rank | String | 段位 |
| wins | Integer | 胜场 |
| losses | Integer | 负场 |
| created_at | DateTime | 注册时间 |

### GameRecord表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer PK | 记录ID |
| player1_id | Integer FK | 玩家1 |
| player2_id | Integer FK | 玩家2/AI |
| winner_id | Integer FK | 赢家 |
| moves | JSON | 落子记录 |
| game_type | String | 对战类型(ai/room) |
| created_at | DateTime | 对局时间 |
